# fullstack pairing exercise

## setup

### client
1. `cd client`
2. `npm install`
3. `npm start`
4. Running on [`localhost:3000`](http://localhost:3000)

### server
1. `cd server`
2. `npm install`
3. `npm start`
5. Running on [`localhost:3001/api`](http://localhost:3001/api)

### testing
- Run `npm test`/`npm test:watch` in `server`
